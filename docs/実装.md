# 依頼：Tabiori-Scrap コア機能の実装

設計書に従い、以下の機能を実装してください。

## 実装ステップ1：環境構築
- `lib/supabase/client.ts` および `server.ts` の作成。
- 環境変数の読み込み設定。

## 実装ステップ2：UIスタイリング (Tailwind CSS)
- `tailwind.config.js` へのカスタムカラー設定。
- スクラップブック風のグローバルスタイルの作成（紙の質感の背景、手書き風フォントの設定）。

## 実装ステップ3：タイムライン画面の実装
- `app/trip/[id]/page.tsx` において、Supabaseから該当する `trip_id` のアイテムを全取得し、時系列（time）順にソートして表示する機能を実装。
- `PLAN` 型と `PHOTO` 型でコンポーネントを出し分けるロジック。

## 実装ステップ4：リアルタイム共同編集
- `supabase.channel()` を使用し、データベースの変更を購読（Subscribe）して、Stateを自動更新するhooksを作成してください。

## 実装ステップ5：画像アップロード
- Supabase Storageへのアップロードと、返却されたURLを `timeline_items` テーブルに保存する一連のアクション。
